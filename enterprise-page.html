Secure and reliable

Encryption

To make it impossible for any third party to interfere with the session and get private information, the data are encrypted in transit and at rest with 256-bit encryption.

Independent Security Audits

We are audited by independent security companies for penetration and vulnerabilities 4 times a year.

Reliability and security

Freedcamp is hosted on Amazon Web Services (AWS) providing end-to-end security and privacy features built-in. Our team deploys best practices for Freedcamp high availability and data recovery.

Permissions

To prevent possible data leaks and access to unauthorized data we provide a flexible set of permissions to make your company's team management easy.

Data safety and availability

Your data and files are safe, encrypted, and securely replicated across multiple regions and the availability zone of AWS. You can automate all data and files downloads daily using our API too.

Systems security

Freedcamp uses Security-Enhanced Linux (SELinux), internet firewalls, and intrusion detection systems on all compute instances.
