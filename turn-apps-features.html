Turn on apps and features

Create a nice, clean workspace for your teams and projects.

Having lots of features and apps can be a good thing, but if you prefer the minimalist look, having them all turned on at once can be overwhelming for not only you, but your teams, customers, stakeholders - anybody who works with you in Freedcamp. We’ve made it easy to find and turn off all the apps and features within Freedcamp, so that you only see what you want to see, whether it’s per project, project group, or globally regardless of the plan you’re on.

For those who manage users

Delegate management of project groups to those you trust … other managers. This way, you’re not a single source for people to talk to about changes. Project group managers can delegate project permissions to those who have permission to manage a project

Projects

Select the account you want to work on. Once you do, you’ll see a list of options along the top related to the project. You’ll be presented with a list of all the available project applications (what’s available depends on your plan). You can use the toggle switch to turn apps and features on and off.

The items listed here are unique to each project. This is features like:

Tasks
Discussions
Milestones
Files

Global modules

Select Global Modules from the main list of items under Manage System. Then choose which apps and features you want to turn off and which you’d like to leave on.

The apps and features under Global Modules are available to all projects in your Freedcamp workspace. These are things like:

Google Drive Integration
Custom Fields
Gantt Charts
